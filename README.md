# Resume Analyzer AI

An AI-powered resume analysis platform designed to help job seekers optimize their resumes for ATS systems, recruiter visibility and interview success.

## Features

* ATS Compatibility Score
* Resume Quality Assessment
* Keyword Gap Analysis
* Skills Evaluation
* Job Description Matching
* AI-Powered Resume Recommendations
* Downloadable Feedback Reports

## Target Users

* Students and Fresh Graduates
* IT Professionals
* Managers and Executives
* Career Changers
* Job Seekers

## Technology Stack

* Python
* OpenAI API
* Streamlit
* PDF Processing
* Natural Language Processing (NLP)
* GitHub
